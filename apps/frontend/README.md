# Walymarket Frontend

React + Vite app using Mysten dApp Kit, Radix Themes, and React Router.

## Scripts

- `pnpm dev` — start dev server
- `pnpm build` — typecheck and build
- `pnpm preview` — preview production build
- `pnpm lint` — run ESLint (if configured at root)

## Network

This app targets Sui testnet only. See `src/networkConfig.ts`.
